# Izzath Tech Store
A premium e-commerce frontend built with **React** for the Syntecxhub internship.

## Features
* **Glassmorphism UI**: Modern, clean aesthetic.
* **Functional Cart**: Add items and calculate totals in AED.
* **Checkout Flow**: Integrated with simulated Tabby/Tamara and Card payments.
* **Order Tracking**: Dynamic tracking status via Order IDs.
